# Student Management System - نظام إدارة الطلاب

نظام متطور لإدارة المؤسسات التعليمية (حضانات، مدارس، معاهد) يعمل مباشرة من المتصفح مع إمكانية التخزين المحلي والمزامنة.

## المميزات
- إدارة بيانات الطلاب بشكل كامل.
- متابعة الأمور المالية والاشتراكات.
- تنبيهات بقرب انتهاء الاشتراكات.
- تصدير واستيراد البيانات (للمزامنة بين الأجهزة).
- دعم الوضع الداكن والفاتح.
- واجهة احترافية باللغة العربية.

## كيف يعمل؟
تم بناء النظام باستخدام HTML, CSS, JavaScript فقط، ويعتمد على **IndexedDB** للتخزين عالي الأداء.

## الاستخدام من أي مكان
1. قم بتصدير نسخة احتياطية من جهازك الحالي.
2. افتح الرابط على الجهاز الجديد.
3. قم باستيراد النسخة الاحتياطية.
